1 ### Using the sms interface
2 
3 The Snorktracker can receive and send SMS using the GSM module.
4 The SMS function must be activated under Settings and
5 and the correct answer phone number must be entered.
6 
7 The following commands are implemented:
8 Note: All commands are case insensitive.
9 
10 #### Switch On or off
11 
12 You can switch the system on or off if you send a
13 **on** or **off** sms to the sim-card phone number.
14 If the system accepted the command it sends a
15 
16  **on -> OK**
17 or
18 
19  **off -> OK**
20 back.
21 
22 #### Send back the status of the system
23 
24 You can query the system for the main information if you send
25 **status** to the Modul phone number.
26 The system should then response with the following result:
27 
28  Status:
29  Sim808 gsm connected
30  Voltage: 12.2V
31  Temperature: 26.5 C
32  Humidity: 40%
33  Pressure: 1006 hPa
34  Altitude: 400 m
35  Speed: 0 kmph
36  Satellites: 5
37  https://maps.google.com/maps?q=61.496052,23.7798"

39 #### Change the 'Power save mode'
40 
41 If your system is in deep sleep it could be very tricky to catch the time when the system awakened.
42 It is much easier to send a 'popwer save mode off' via sms to the module.
43 To do that you have to send a **psm:off** to the system phone number and wait for a successful response
44 
45  **psm:off -> OK**
46 
47 Afterwards you can switch on the power save mode via the web interface or by sending a
48 **psm** sms.
49 
50  **psm -> OK**
51 
52 #### Ask for the current gps position
53 
54 If you want to know the gps position of your system you can send a simple sms with **gps**.
55 If the gps awake the next time and the gps tracking is active then you will receive a URL
56 with the current gps position as an google request.
57 
58 This URL can then easily be opened via your browsers in google and if everything works you
59 will see the current location of the module on a Google map
60 
61  https://maps.google.com/maps?q=61.496052,23.7798"
62 
63 #### Set some sms parameter
64 
65 You can change the interval at which the system polls incoming SMS.
66 For this you can send i.e. the following command
67 
68 **sms:15**
69 
70 This change the interval to 15 seconds.
71 The response should be
72 
73  sms:15 -> OK
74 
75 #### Set some mqtt parameter
76 
77 If you want to change the mqtt interval for sending data on moving or standing you can use the command sms
78 **mqtt:30:60**
79 This sets the interval on moving to 30 seconds and the interval on not moving to 30 seconds.
80 The response should be
81 
82  mqtt:30:60 -> OK
83 
84 ### Change the response phone number
85 
86 If you want to change the phone number to which the result should go then you can do this with:
87 
88 **phone:123456789**
89 
90 Then the new mobile phone number should receive the following result sms.
91 
92  phone:123456789 -> OK
